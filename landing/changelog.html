  • - Hotfix: se solucionaron varios bugs visuales propios de la Steam Store

Steamcito v0.41 (25/11/2020)

  • - Hotfix: se solucionó bug de desfasaje de posición de los precios en packages /sub/ Ver issue

Steamcito v0.4 (24/11/2020)

  • - Nueva funcionalidad: Steamcito ahora funciona en todas las ventanas de carga de saldo. Ver issue
  • - Hotfix: se solucionó bug de que Steamcito no mostraba precios en Bundles al estar deslogueado. Ver issue
  • - Enhancement: Ya no se te abre una nueva pestaña con el changelog cada vez que Steamcito se actualiza o instala. Ahora sólo se te muestra una notificación en la home de Steam por única vez.
  • - Enhancement: Mejora de Iconos en el carrito

Steamcito v0.3 (15/11/2020)

  • - Se implementó compatibilidad con sistemas operativos que no cuentan con Unicode 12.0 (Ej: Windows 7, Windows 8, Ubuntu) para mostrar correctamente el emoji del mate en estos sistemas operativos.
  • - Se implementó apertura automática del changelog tras instalación o actualización.

Steamcito v0.2 (12/11/2020)

  • - Se corrigió bug en el cual no se mostraba el porcentaje del descuento en las DLCs que tenían descuento.
